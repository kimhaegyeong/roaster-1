[github 사용법]



[github clone]
깃헙에 있는 소스를 복사한다.

복사할 경로
gimhaegyeong-ui-MacBook-Pro:~ schema9$ pwd
/Users/schema9

git의 경로
gimhaegyeong-ui-MacBook-Pro:~ schema9$ which git
/usr/bin/git

복사할 github의 경로를 입력하고 명령어를 실행하면
소스를 복사해온다.
gimhaegyeong-ui-MacBook-Pro:~ schema9$ git clone https://github.com/kimhaegyeong/roaster-1
